Agreement manager

O aplikacji

Wykorzystanie technologie:

Podstawowe:

Java 8, Spring MVC 5.0.6, Hibernate 5.2.17, Apache Maven 3.3.9, Apache Tomcat 8.0.52, Apache Tiles 3.0.8, Apache POI 3.17

Webowe:

HTML5, CSS3, Javascript, jQuery UI, jQuery DataTable, jQuery UI File Upload

Bazy danych:

Aplikacja powstała przy pomocy bazy danych Oracle 12c, a ponadto została przetstowana we współpracy z bazą danych PostgreSQL 10.4.

Ponadto :

Kodowanie miało miejsce pod kontrolą systemu Manjaro Linux 17.1 oraz środowiska programistycznego Intellij IDEA Ultimate2018.1.3. Stronę wizualną projektu przygotowano przy pomocy programu Webstorm 2018.1.2. Działanie aplikacji sprawdzono także pod kontrolą systemu Windows 10 z zainstalowanym środowiskiem programistycznym Eclipse.

Testy nie pokazały niewłaściwego działania aplikacji.

Dokumentacja została stworzona za pomocą programu Javadoc oraz języka Latex2e.
